✨ Alpine.js 优势

  • ✅ 响应式数据绑定：状态变化自动更新 UI，无需手动操作 DOM
  • ✅ 代码简洁：相比 jQuery 减少 60% 代码量
  • ✅ 易于维护：声明式语法，逻辑清晰
  • ✅ 体积小巧：仅 15KB，加载速度快
  • ✅ 渐进式集成：可与现有项目无缝结合

🎮 交互说明

  • 🖱️ 单击宠物：增加亲密度，随机反应
  • 🖱️🖱️ 双击宠物：旋转动画
  • 🍖 喂食：降低饥饿度，恢复少量能量
  • ❤️ 抚摸：增加亲密度，改变心情
  • 🎾 玩耍：消耗能量，大幅增加亲密度
  • 💤 睡觉：恢复能量，离线时间计算

📝 使用方法

  1. 在 HTML 中引入 Alpine.js CDN：
    <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>
  2. 使用 x-data 指令初始化组件
  3. 使用 x-text、:class、@click 等指令绑定数据和事件
  4. 状态自动持久化到 localStorage
