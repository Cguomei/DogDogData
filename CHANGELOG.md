## 版本说明

### 版本号格式
- **主版本号** - 不兼容的 API 修改
- **次版本号** - 向下兼容的功能性新增
- **修订号** - 向下兼容的问题修正

### 标签类型
- `新增` - 新功能
- `改进` - 现有功能的改进
- `修复` - Bug 修复
- `废弃` - 即将移除的功能
- `移除` - 已移除的功能
- `安全` - 安全相关的修复
