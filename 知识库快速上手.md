## 🔧 如何添加新知识

### 方法1: 直接编辑JSON文件

1. 打开 `data/knowledge_base.json`
2. 添加新的知识条目：

```json
{
  "knowledge": {
    "如何给狗狗洗澡": {
      "question_patterns": ["怎么.*洗澡", "如何.*洗澡", "洗澡.*步骤"],
      "answer": "🛁 **狗狗洗澡步骤**\n\n1. 准备温水...\n2. ...",
      "category": "general_qa",
      "confidence": 0.9,
      "created_at": "2026-05-09T19:00:00",
      "usage_count": 0
    }
  }
}
```

3. 保存文件
4. 重启Flask应用

### 方法2: 通过代码添加

创建脚本 `add_knowledge.py`：

```python
from utils.knowledge_base import get_knowledge_base
from datetime import datetime

kb = get_knowledge_base()

# 添加新知识
kb.add_knowledge("训练握手", {
    "question_patterns": ["怎么.*握手", "训练.*握手"],
    "answer": "✋ **训练狗狗握手**\n\n步骤：\n1. 准备零食\n2. 说'握手'\n3. 轻抬前爪\n4. 成功后奖励",
    "category": "general_qa",
    "confidence": 0.9,
    "created_at": datetime.now().isoformat(),
    "usage_count": 0
})

# 保存
kb.save()
print("✅ 知识添加成功")
```

运行：
```bash
python add_knowledge.py
```

## 🔍 故障排查

### Q1: 知识库没有生效？

**检查步骤**：
1. 查看日志：`log/ai_assistant.log`
2. 搜索关键词："知识库命中"或"知识库未命中"
3. 确认`data/knowledge_base.json`存在

**常见原因**：
- 问题模式不匹配
- 类别指定错误
- JSON文件格式错误

### Q2: 如何提高命中率？

**方法**：
1. 分析日志中的未命中问题
2. 为这些问题添加知识
3. 扩充question_patterns（支持正则）

示例：
```json
{
  "question_patterns": [
    "泰迪.*特点",
    "泰迪.*怎么样", 
    "泰迪犬.*介绍",
    "贵宾犬.*特征"
  ]
}
```

**位置**：`D:\PycharmProjects\fastApiProject\data\knowledge_base.json`

可以直接编辑这个文件来修改知识。
