## 版本说明

### 版本号规则

采用语义化版本 `MAJOR.MINOR.PATCH`：

- **MAJOR**：不兼容的 API 变更或重大功能调整
- **MINOR**：向后兼容的功能新增
- **PATCH**：向后兼容的问题修正

### 类型标识

- ✨ **新增**：新功能或重大改进
- 🐛 **修复**：Bug 修复
- 🔧 **改进**：技术优化或重构
- 📝 **文档**：文档更新
- 🔒 **安全**：安全加固
- 📊 **测试**：测试相关
- ⚡ **性能**：性能优化
- ♻️ **重构**：代码重构
