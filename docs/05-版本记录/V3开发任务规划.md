**技术要点**:
```python
# 在调用模型时传入历史消息
def call_model_with_context(user_message, session_id):
    # 1. 获取最近10条历史消息
    history = ChatMessage.query.filter_by(
        session_id=session_id
    ).order_by(
        ChatMessage.created_at.desc()
    ).limit(10).all()
    
    # 2. 反转顺序（从旧到新）
    history.reverse()
    
    # 3. 构建完整的消息列表
    messages = [
        {"role": msg.role, "content": msg.content}
        for msg in history
    ]
    
    # 4. 添加当前用户消息
    messages.append({"role": "user", "content": user_message})
    
    # 5. 调用模型
    return call_local_model(messages)
```

**验收标准**:

#### 任务2: 自动学习机制 ⭐⭐⭐⭐⭐

**优先级**: P0  
**预计工时**: 3天  
**负责人**: AI工程师

**需求描述**:
将模型的优质回答自动加入知识库，减少重复调用模型，提升响应速度。

**技术要点**:
```python
def auto_learn_from_answer(question, answer, question_type, confidence=0.9):
    """从模型回答中自动学习"""
    if confidence > 0.8:  # 高置信度才学习
        kb = get_knowledge_base()
        
        # 检查是否已存在相似问题
        similar = kb.search_similar(question, threshold=0.85)
        if not similar:
            # 添加到知识库
            kb.add_knowledge(
                question=question,
                answer=answer,
                category=question_type,
                source='model',
                confidence=confidence
            )
            
            logger.info(f"✅ 自动学习: {question[:50]}...")
            return True
    
    return False
```

**学习策略**:
1. **高置信度自动学习**（>0.8）- 直接加入知识库
2. **中等置信度待审核**（0.6-0.8）- 标记为待审核
3. **低置信度不学习**（<0.6）- 忽略

**验收标准**:
- [ ] 模型生成的优质回答自动保存到知识库
- [ ] 避免重复学习相同问题
- [ ] 知识库命中率提升20%以上
- [ ] 平均响应时间降低30%

**相关文件**:
- `utils/knowledge_base.py` - 添加auto_learn方法
- `routes/ai_assistant.py` - 在chat函数中调用学习

```python
from apscheduler.schedulers.background import BackgroundScheduler

def check_price_alerts():
    """检查价格异常"""
    breeds = DogBreed.query.all()
    
    for breed in breeds:
        current_avg = get_current_avg_price(breed.name)
        last_week_avg = get_last_week_avg_price(breed.name)
        
        change_rate = (current_avg - last_week_avg) / last_week_avg
        
        if change_rate > 0.15:  # 涨幅超过15%
            send_alert(
                type="price_increase",
                breed=breed.name,
                change_rate=change_rate,
                message=f"⚠️ {breed.name}价格上涨{change_rate*100:.1f}%"
            )

# 每天凌晨2点执行
scheduler = BackgroundScheduler()
scheduler.add_job(check_price_alerts, 'cron', hour=2)
scheduler.start()
```

**验收标准**:
