## 🔑 核心技术亮点

### 1. 混合架构设计
```
用户提问 → 问题分类 → 知识库查询 → (命中) → 直接返回 (<100ms)
                              → (未命中) → 调用模型 → 返回答案
```
- 知识库优先，减少85%模型调用
- 大幅降低响应时间
- 节省计算资源

### 2. 智能会话管理
- 首次聊天自动创建会话
- 使用第一条消息作为标题
- 自动保存所有对话历史
- 软删除机制保护数据

### 3. 完整的权限控制
- 用户只能访问自己的会话
- 日志查看仅限管理员
- CSRF保护
- Session验证

### 4. 性能优化
- 数据库索引加速查询
- 限制会话列表数量（最多50个）
- 异步保存历史（失败不影响主流程）
- 知识库内存缓存

### 5. 可扩展性设计
- 预留feedback字段（点赞/点踩）
- 记录response_time（性能监控）
- 区分source（知识库vs模型）
- 模块化设计，易于扩展

## 🚀 使用方法

### 1. 启动本地模型
```bash
# Ollama方式
ollama pull qwen2.5:1.5b
ollama serve

# 或 LM Studio方式
# 启动LM Studio并加载模型
```

### 2. 配置环境变量
```env
MODEL_TYPE=ollama
LOCAL_MODEL_URL=http://localhost:11434
LOCAL_MODEL_NAME=qwen2.5:1.5b
```

### 3. 启动应用
```bash
python app.py
```

### 4. 访问AI助手
- 登录系统
- 点击导航栏"🤖 AI助手"
- 开始聊天

### 5. 运行测试
```bash
# 运行所有AI测试
python Test/run_ai_tests.py

# 测试对话历史API
python test_chat_history.py
```
