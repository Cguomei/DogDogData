### 4. 提示词工程优化

#### 4.1 通用问答优化

**修改函数**: `handle_general_qa()`

**优化前**:
```python
{"role": "system", "content": "你是宠物知识专家，回答要简洁专业"}
```

**优化后**:
```python
system_prompt = """你是宠物知识专家，拥有丰富的狗狗养护、训练、健康等专业知识。

回答要求：
1. 专业准确：基于科学知识和实践经验
2. 结构清晰：使用列表、分段等方式组织内容
3. 实用性强：给出可操作的建议
4. 语气友好：亲切自然，易于理解
5. 长度适中：100-300字为宜
6. 安全第一：涉及健康问题时提醒咨询兽医

请根据用户问题提供专业、实用的回答。"""
```

**改进点**:
- ✅ 明确角色定位
- ✅ 规范回答格式
- ✅ 强调实用性
- ✅ 控制回答长度
- ✅ 注重安全性

---

#### 4.2 品种信息优化

**修改函数**: `get_breed_info_from_model()`

**优化前**:
```python
prompt = f"请介绍一下{breed}犬的特点，包括性格、体型、养护难度等。"
```

**优化后**:
```python
prompt = f"""请详细介绍{breed}犬，包括以下方面：

1. **基本特征**：体型、毛色、寿命等
2. **性格特点**：温顺度、活跃度、智商等
3. **养护要点**：运动需求、梳理频率、饮食建议
4. **训练建议**：难易程度、注意事项
5. **适合人群**：新手/有经验、家庭/单身等

请用清晰的结构呈现，语气友好专业。"""
```

**改进点**:
- ✅ 结构化输出要求
- ✅ 覆盖全面维度
- ✅ 明确内容要点
- ✅ 规范表达风格
