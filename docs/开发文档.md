# 项目开发文档

## 项目概述

### 项目名称
狗狗数据分析系统 v2.4.0

### 项目简介
本项目是一个基于 Flask 的狗狗数据分析平台，提供数据可视化、品种管理、狗粮数据查询等功能。通过 PyECharts 生成丰富的图表，帮助分析宠物市场数据。

**最新版本**: v2.4.0 - 图表列表页面、体型分布图修复、导航优化

### 技术栈

#### 后端框架
- **Flask 3.1.3**: Web 应用框架
- **Flask-SQLAlchemy 3.1.1**: ORM 数据库操作
- **Flask-Login 0.6.3**: 用户认证管理
- **Flask-WTF 1.2.2**: 表单处理和 CSRF 保护
- **Flask-Caching 2.3.1**: 缓存支持
- **Flask-APScheduler 1.13.1**: 定时任务调度

#### 数据库
- **MySQL**: 关系型数据库
- **PyMySQL 1.1.2**: MySQL Python 驱动

#### 数据可视化
- **PyECharts 2.1.0**: ECharts Python 库
- **Pandas 2.3.3**: 数据处理和分析

#### 安全相关
- **Werkzeug**: 密码哈希加密
- **python-dotenv 1.2.2**: 环境变量管理

#### 测试工具
- **pytest 9.0.2**: 测试框架
- **pytest-flask 1.13.0**: Flask 测试支持
- **Selenium 4.41.0**: UI 自动化测试

## 核心功能模块

### 1. 用户认证模块

#### 功能描述
- 用户注册、登录、登出
- 基于角色的权限控制（管理员/普通用户）
- Session 管理

#### API 接口
```
POST /register          # 用户注册
POST /login            # 用户登录
GET  /logout           # 用户登出
```

#### 安全特性
- 密码加密存储（Werkzeug）
- CSRF 保护（Flask-WTF）
- 用户名格式验证
- 密码强度检查（至少 6 位）
- 防 SQL 注入（ORM 参数化查询）
- 防枚举攻击（不明确的错误提示）

### 2. 品种管理模块

#### 功能描述
- CRUD 操作（创建、读取、更新、删除）
- 数据导入（CSV/Excel）
- 数据验证和过滤

#### API 接口
```
GET    /api/breeds           # 获取所有品种
POST   /api/breeds           # 创建新品种
GET    /api/breeds/<id>      # 获取单个品种
PUT    /api/breeds/<id>      # 更新品种
DELETE /api/breeds/<id>      # 删除品种
POST   /api/breeds/import    # 批量导入
```

#### 数据验证规则
- **breed_name**: 必填，2-100 字符，禁止 HTML 标签
- **avg_life_years**: 可选，0-100 岁，数值类型
- **size_category**: 可选，枚举值 [小型，中型，大型，超大型]
- **popularity**: 可选，0-1000，整数

### 3. 数据可视化模块

#### 图表类型
1. **价格散点图** (`/chart/scatter`)
   - 展示不同价格区间的狗狗数量分布
   
2. **体重折线图** (`/chart/line`)
   - 展示不同体重区间的趋势变化
   
3. **级别柱状图** (`/chart/bar`)
   - 对比宠物级和血统级的数量
   
4. **TOP10 直方图** (`/chart/hist`)
   - 展示最热门的 10 种狗狗和店铺
   
5. **价格段漏斗图** (`/chart/funnel`)
   - 展示各价格区间的分布情况
   
6. **世界地图** (`/chart/map`)
   - 展示狗狗家乡分布（带翻译功能）

#### 数据来源
- 数据库表：`jd_dogs`, `dog_price`, `dog_wykl`
- 定时任务每 6 小时更新汇总数据到 `dashboard_summary` 表

### 4. 狗粮数据模块

#### 功能描述
- 狗粮品牌统计
- 价格区间分析
- 产地分布统计

#### API 接口
```
GET /food      # 狗粮数据看板
GET /api/food  # 狗粮原始数据
```

## 数据库设计

### 表结构

#### 1. users (用户表)
```sql
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(80) UNIQUE NOT NULL,
    password_hash VARCHAR(200) NOT NULL,
    role VARCHAR(20) DEFAULT 'user',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 2. dog_breeds (品种表)
```sql
CREATE TABLE dog_breeds (
    id INT PRIMARY KEY AUTO_INCREMENT,
    breed_name VARCHAR(100) UNIQUE NOT NULL,
    avg_life_years DECIMAL(3,1),
    size_category ENUM('小型', '中型', '大型', '超大型'),
    popularity INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### 3. dashboard_summary (数据汇总表)
```sql
CREATE TABLE dashboard_summary (
    id INT PRIMARY KEY,
    total_dogs INT,
    avg_price DECIMAL(10,2),
    total_shops INT,
    total_breeds INT,
    top_breeds JSON,
    top_shops JSON,
    price_dist JSON,
    level_pet INT,
    level_blood INT,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

## 环境配置

### 开发环境配置

#### 1. 安装依赖
```bash
pip install -r requirements.txt
```

#### 2. 配置环境变量 (.env 文件)
```env
DB_HOST=localhost
DB_USER=doguser
DB_PASSWORD=123456
DB_NAME=dog
SECRET_KEY=你的随机密钥
```

#### 3. 初始化数据库
```bash
python -c "from app import app; from models import db; \
with app.app_context(): db.create_all()"
```

#### 4. 创建管理员账户

```python
from app_new import app, db
from models import User

with app.app_context():
    admin = User(username='admin', role='admin')
    admin.set_password('123456')
    db.session.add(admin)
    db.session.commit()
```

#### 5. 启动开发服务器
```bash
python app_new.py
```

访问地址：http://localhost:5000

## 安全规范

### 输入验证
- ✅ 所有用户输入必须验证类型、长度、格式
- ✅ 使用白名单验证（只允许已知安全的字符）
- ✅ 数值范围检查
- ✅ 枚举值检查

### 输出编码
- ✅ Jinja2 模板自动转义
- ✅ 禁止直接渲染用户输入的 HTML
- ✅ JSON 响应设置正确的 Content-Type

### 认证授权
- ✅ 密码加密存储
- ✅ Session 安全管理
- ✅ CSRF 保护
- ✅ 敏感操作需要重新认证

### 数据安全
- ✅ ORM 参数化查询（防 SQL 注入）
- ✅ 错误信息不泄露敏感数据
- ✅ 数据库连接信息放在环境变量

### 响应头安全
```python
X-Frame-Options: SAMEORIGIN
X-Content-Type-Options: nosniff
X-XSS-Protection: 1; mode=block
Referrer-Policy: strict-origin-when-cross-origin
```

## 部署指南

### 生产环境配置

#### 1. 关闭 Debug 模式
```python
app.run(debug=False)
```

#### 2. 使用 Gunicorn
```bash
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

#### 3. 配置 Nginx 反向代理
```nginx
server {
    listen 80;
    server_name yourdomain.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

#### 4. 启用 HTTPS
```bash
# 使用 Let's Encrypt 免费证书
certbot --nginx -d yourdomain.com
```

## 常见问题

### Q1: 如何重置管理员密码？

```python
from app_new import app, db
from models import User

with app.app_context():
    user = User.query.filter_by(username='admin').first()
    user.set_password('new_password')
    db.session.commit()
```

### Q2: 数据汇总表如何更新？
手动触发:
```python
from charts import update_dashboard_summary
update_dashboard_summary()
```

自动更新：定时任务每 6 小时执行一次

### Q3: 如何处理大文件导入？
当前版本限制单次导入不超过 1000 条记录，大批量数据建议分批导入。
