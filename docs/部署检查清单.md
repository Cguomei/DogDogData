## 六、环境变量准备


## 常见问题预案

### 问题 1：部署失败

**解决**：查看部署日志，常见原因：
- 依赖安装失败（检查 requirements.txt）
- 数据库连接失败（检查 DATABASE_URL）
- 环境变量缺失（检查 Variables）

### 问题 2：500 错误

**解决**：
- 查看应用日志
- 检查数据库是否初始化
- 验证环境变量配置

### 问题 3：访问速度慢

**解决**：
- 这是正常的（国外服务器）
- 考虑使用 Cloudflare CDN
- 或迁移到国内云服务器

### 问题 4：数据丢失

**解决**：
- 检查数据库服务状态
- 从备份恢复
- 重新运行初始化脚本
