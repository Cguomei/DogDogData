### 更新时机

- ✅ 代码提交前：确保文档与代码同步
- ✅ 版本发布时：创建版本发布说明
- ✅ 功能完成后：更新相关技术文档
- ✅ 问题修复后：记录到修复日志

### 版本号规则

- 主版本.次版本.修订号 (如: v4.3.1)
- 所有文档版本号必须统一
