# 狗狗数据分析系统 — 产品说明

## 概述

狗狗数据分析系统是一站式宠物犬数据服务平台，聚合多渠道狗狗市场数据（价格、品种、销量），提供数据看板、AI 智能助手、价格监控告警等能力。面向养宠用户、爱宠人士、宠物行业从业者。

## 核心能力

### 1. 数据看板
首页聚合展示关键指标：品种总数、平均价格、最高价格、数据来源渠道数。内置定时任务每 6 小时自动刷新摘要。

### 2. 品种管理
管理员可对狗狗品种进行完整的 CRUD 操作，维护品种名称、平均寿命、体型分类、人气值等字段。

### 3. 数据图表（PyECharts）
- **散点图** — 价格分布
- **折线图** — 体重趋势
- **柱状图** — 级别统计
- **TOP10 直方图** — 热门品种/店铺排行
- **漏斗图** — 价格区间转化分析
- **世界地图** — 狗狗产地分布

### 4. AI 智能助手
内置本地 LLM（Ollama / LM Studio）驱动的智能问答系统：
- **价格查询** — "金毛多少钱？"
- **品种介绍** — "介绍边境牧羊犬"
- **购买推荐** — "预算 5000，推荐什么狗？"
- **品种对比** — "金毛和拉布拉多哪个好？"
- **智能图表生成** — 根据对话上下文自动生成图表
- **知识库学习** — 支持人工录入 + 自动学习高质量回答
- **会话管理** — 多轮对话、历史记录、反馈评价

### 5. 价格与品种告警
- 设置目标价格，低于/高于阈值时通知
- 订阅新品种上架通知
- 未读消息计数与已读管理

### 6. 用户偏好
支持个性化设置：偏好品种、体型、预算范围、养宠经验、AI 回复风格等，驱动精准推荐。

### 7. 用户反馈
用户可提交 bug/功能建议/改进意见，管理员可回复和处理，支持优先级和状态流转。

### 8. 用户行为分析
记录页面浏览、按钮点击等事件，支持漏斗分析和设备/浏览器维度统计。

### 9. 国际化
支持中文、英文、日文，通过 Flask-Babel 实现，URL 参数 / Session / 请求头三级探测。

### 10. 自定义分析
上传 CSV/Excel 文件，系统自动进行数据质量校验（空值检测、类型异常、重复值），支持自定义图表生成（散点/折线/柱状/饼图）。

## 用户角色

| 角色 | 能力 |
|------|------|
| **游客** | 浏览首页、图表、品种列表、AI 聊天（访客模式） |
| **注册用户** | 所有游客能力 + 提交反馈、设置偏好与告警、查看 AI 历史记录 |
| **管理员** | 所有用户能力 + 品种 CRUD、查看反馈列表并回复、查看 AI 日志、数据统计面板 |

## 数据来源

多渠道聚合：京东、淘宝/天猫、58 同城、狗铺子等公开数据源的狗狗价格与品种信息。

## 技术栈速览

| 层 | 技术 |
|----|------|
| 后端框架 | Flask 3.1 + 10 蓝图 |
| ORM | SQLAlchemy 2.0 + PyMySQL |
| 数据库 | MySQL 8.0 |
| 前端渲染 | Jinja2 + Alpine.js + PyECharts |
| AI | Ollama / LM Studio + 本地知识库 |
| 国际化 | Flask-Babel (zh_CN / en_US / ja_JP) |
| 监控 | Prometheus + Grafana (docker-compose) |
| 认证 | Flask-Login (Session) + PyJWT (Token) |
| 试验项目 | FastAPI 试点 (v5.0.0-alpha, port 8000, SQLite) |
| Vue 前端 | Vue 3 + Element Plus + Vite (port 3000, 对接 FastAPI) |
