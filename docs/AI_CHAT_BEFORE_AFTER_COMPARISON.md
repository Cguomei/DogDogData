## 7. 技术实现对比

### CSS布局

**修复前：**
```css
.chat-header {
    text-align: center;  /* 所有元素居中 */
    padding: 20px;       /* 内边距过大 */
}
/* 结果：垂直堆叠，占用空间大 */
```

**修复后：**
```css
.chat-header {
    display: flex;                    /* Flexbox布局 */
    justify-content: space-between;   /* 左右分布 */
    align-items: center;              /* 垂直居中 */
    padding: 15px 20px;               /* 减小内边距 */
}
/* 结果：水平排列，节省空间 */
```

---

### JavaScript错误处理

**修复前：**
```javascript
// 简单粗暴
try {
    const response = await fetch(url);
    const data = await response.json();
    // 直接使用，不检查状态
} catch (error) {
    console.error('加载失败');  // 模糊的错误信息
}
```

**修复后：**
```javascript
// 精细化处理
try {
    const response = await fetch(url);
    
    if (response.status === 401) {
        // 未登录处理
        return;
    }
    
    if (response.status === 404) {
        // 不存在处理
        return;
    }
    
    if (!response.ok) {
        // 其他HTTP错误
        console.error(`HTTP ${response.status}`);
        return;
    }
    
    const data = await response.json();
    // 安全使用数据
    
} catch (error) {
    console.error('网络错误:', error);
    // 不清除localStorage，可能是临时问题
}
```
