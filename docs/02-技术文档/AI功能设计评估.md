## 🔧 关键问题分析

### 问题1: 图表不够智能 ❌

**现状：**
```python
# 当前：返回固定图表
def handle_chart_generation(params, context):
    if chart_type == 'price_scatter':
        html = charts.get_price_scatter()  # 所有品种的固定数据
        return f"<div>{html}</div>"
```

**问题：**
- 用户问"金毛的价格分布"，却看到所有品种的散点图
- 无法根据问题动态过滤数据
- 图表参数不可配置

**建议改进：**
```python
def handle_chart_generation(params, context):
    breed = params.get('breed') or extract_breed_from_context(context)
    
    if chart_type == 'price_scatter':
        # 动态查询指定品种的数据
        data = query_price_data(breed=breed)
        html = charts.get_price_scatter(data=data, title=f"{breed}价格分布")
        return f"<div>{html}</div>"
```

### 问题4: 安全问题需加强 ⚠️

**现状：**
