```python
def test_create_breed_validation():
```

### 3.1 CSRF 保护

**测试要点**:
```markdown
✅ 表单提交必须带 CSRF token
- 登录表单
- 注册表单
- 品种管理表单

✅ API 接口（JSON）豁免
- Content-Type: application/json
- 不需要 CSRF token
```

**测试用例**:
```python
def test_csrf_protection():
    # 不带 CSRF token 提交表单
    response = client.post('/login', data={
        "username": "admin",
        "password": "123456"
        # 缺少 csrf_token
    })
    assert response.status_code == 403  # 应该被拒绝
    
    # 带 CSRF token
    response = client.get('/login')
    csrf_token = extract_csrf_token(response.data)
    
    response = client.post('/login', data={
        "username": "admin",
        "password": "123456",
        "csrf_token": csrf_token
    })
    assert response.status_code == 302  # 重定向成功
```

```python
from locust import HttpUser, task, between

class DogAnalysisUser(HttpUser):
    wait_time = between(1, 3)
    
    @task(3)
    def view_index(self):
        self.client.get("/")
    
    @task(2)

        charts = [
            "/chart/scatter",
            "/chart/line",
            "/chart/bar",
            "/chart/hist",
            "/chart/funnel",
            "/chart/map"
        ]
        for chart in charts:

    def api_breeds(self):

    @task(1)
    def login_logout(self):

        self.client.post("/login", {
```

## 十、常见问题 FAQ

### Q1: 测试数据哪里来？

**A**: 
1. 使用 `init_db.py` 初始化测试数据库
2. 从生产环境脱敏导出
3. 手动构造边界数据

### Q2: 如何测试定时任务？

**A**:
```python
# 手动触发定时任务
from charts import update_dashboard_summary
update_dashboard_summary()

# 或者修改调度频率
scheduler.remove_job('update_summary')
scheduler.add_job(update_dashboard_summary, 'interval', seconds=10)
```

### Q3: 缓存如何清除？

**A**:
```python
# 清除所有缓存
cache.clear()

# 清除特定缓存
cache.delete_memoized('get_world_map')

# 访问清除缓存页面
GET /clear-cache
```

### Q4: 宠物功能不显示怎么办？

**A**:
1. 检查浏览器控制台是否有 JS 错误
2. 清除浏览器缓存
3. 确认 localStorage 可用
4. 检查 pet.js 是否正确加载

### Q5: 如何模拟高并发？

**A**:
```bash
# 使用 Locust
locust -f test.py --host=http://localhost:5000 -u 100 -r 10

# 使用 JMeter
jmeter -n -t test_plan.jmx -l results.jtl
```
