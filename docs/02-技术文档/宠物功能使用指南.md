## 🐶 如何与小宠物互动

### 基本互动
1. **点击宠物身体**: 宠物会跳动并显示随机反应
2. **鼠标悬停**: 宠物会主动和你说话
3. **动作按钮**（鼠标悬停时显示）:
   - 🍖 **喂食**: 降低饥饿度
   - 🎾 **玩耍**: 增加亲密度，但消耗活力
   - ❤️ **抚摸**: 让宠物开心
   - 💤 **睡觉**: 切换睡眠状态

### 状态查看
宠物下方有两个状态条：
- **上方（绿色→黄色→红色）**: 饥饿度（越短越饿）
- **下方（绿色→黄色→红色）**: 活力值（越短越累）

### 智能功能
- ⏱️ **自动隐藏**: 60 秒无操作会自动隐藏，右下角显示召唤按钮 🐶
- 💾 **状态保存**: 关闭浏览器也会保存宠物状态
- 🌙 **离线计算**: 再次访问时，宠物会根据你离开的时间变饿或变累
- 😊 **心情系统**: 宠物会根据状态显示不同表情

## 🔧 自定义配置

### 更改宠物类型

在页面的 JavaScript 中添加：

```javascript
document.addEventListener('DOMContentLoaded', () => {
    window.virtualPet = new VirtualPet({
        petName: '喵喵',  // 自定义名称
        petType: 'cat',   // dog, cat, rabbit
        autoHideTimeout: 30000  // 30 秒后自动隐藏
    });
});
```

### 禁用宠物

如果某个页面不需要宠物，可以在该页面的模板中：

```html
{% block extra_scripts %}
<script>
// 阻止宠物初始化
window.disableVirtualPet = true;
</script>
{% endblock %}
```

## 🐛 常见问题

### Q: 宠物不显示怎么办？
A: 
1. 检查浏览器控制台是否有错误
2. 清除浏览器缓存
3. 确认 JavaScript 已启用

### Q: 如何重置宠物状态？
A: 打开浏览器开发者工具，执行：
```javascript
localStorage.removeItem('virtualPetState');
location.reload();
```

### Q: 宠物太遮挡内容怎么办？
A: 可以调整位置：
```css
.virtual-pet-container {
    bottom: 10px;
    right: 10px;
    transform: scale(0.8);
}
```

### Q: 支持哪些浏览器？
A: 
- ✅ Chrome / Edge (推荐)
- ✅ Firefox
- ✅ Safari
- ⚠️ IE11 (部分功能受限)

## 📊 测试建议

### 测试步骤
1. 访问任意页面，确认宠物显示
2. 点击宠物，观察动画和对话
3. 使用动作按钮，检查状态变化
4. 等待 60 秒，验证自动隐藏
5. 点击召唤按钮，确认重新出现
6. 关闭浏览器重新打开，验证状态保存

### 性能测试
1. 打开多个标签页（宠物只在活动标签显示）
2. 长时间运行（检查内存泄漏）
3. 快速点击（检查动画流畅度）
