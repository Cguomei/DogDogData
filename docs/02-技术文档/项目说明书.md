# 狗狗数据分析系统 - 项目说明书

**项目名称**: 狗狗数据分析系统 (Dog Data Analysis System)  
**当前版本**: v4.5.3  
**技术栈**: Flask + MySQL + PyECharts + Playwright  
**开发团队**: Lingma AI Assistant  
**最后更新**: 2026-04-20

---

## 📖 项目概述

狗狗数据分析系统是一个功能完善的数据可视化 Web 应用平台，专注于狗狗品种、狗粮、价格等数据的分析与展示。系统采用 Flask 框架构建，提供丰富的图表展示、用户认证、品种管理等功能，并创新性地集成了网页小宠物互动系统。

### 核心价值
- 📊 **数据可视化** - 6 种图表类型，直观展示数据分析结果
- 🐶 **网页小宠物** - 可交互的虚拟宠物，提升用户体验
- 🔐 **用户认证** - Session + JWT 双重认证机制
- 📱 **RESTful API** - 标准化的 API 接口，支持移动端扩展
- 🧪 **自动化测试** - 完善的分层测试体系（API + UI + E2E）

## 🎯 功能模块

### 1. 数据看板
**功能描述**: 首页展示 8 项核心指标，提供数据概览

**核心指标**:
- 品种总数
- 平均寿命
- 价格区间
- 体型分布
- 热门品种 TOP10
- 地域分布
- 用户数量
- 收藏数量

**技术实现**:
- Flask 路由: `/`
- 数据来源: MySQL 数据库
- 缓存策略: Flask-Caching

---

### 2. 数据可视化
**功能描述**: 提供 6 种图表类型，支持多维度数据分析

**图表类型**:
1. **散点图** - 价格分布分析
2. **折线图** - 体重趋势变化
3. **柱状图** - 级别对比分析
4. **直方图** - TOP10 热门排行
5. **漏斗图** - 转化流程分析
6. **地图** - 地域分布展示（支持中文地名自动翻译）

**技术实现**:
- 图表库: PyECharts 2.1.0
- 路由: `/charts`, `/charts/list`
- 特性: 响应式布局、交互提示、数据导出

---

### 3. 自定义数据分析
**功能描述**: 用户上传 CSV/Excel 文件，自动生成图表

**核心功能**:
- 📤 文件上传（CSV/Excel）
- ✅ 数据质量校验（空值、异常值、重复值检测）
- 📊 4 种图表类型选择（散点图、折线图、柱状图、饼图）
- 💾 数据导出（CSV/Excel）

**技术实现**:
- 路由: `/custom-analysis`
- API: `/api/upload-data`, `/api/generate-chart`, `/api/export-data`
- 数据处理: Pandas 2.3.3
- Excel 处理: openpyxl 3.1.5

---

### 4. 网页小宠物系统
**功能描述**: 可交互的虚拟宠物，陪伴用户浏览

**特色功能**:
- 🐶 **可爱互动** - 触摸、喂食、玩耍、爱心、睡觉
- 💖 **情感系统** - 开心、困倦、不开心等丰富表情
- ⚡ **状态持久** - localStorage 本地存储，自动保存进度
- 🎨 **精美动画** - 8 种 CSS 动画效果，60fps 流畅体验
- 📱 **全平台支持** - 桌面端和移动端完美适配
- 🖱️ **拖动功能** - 可拖动宠物位置，避免遮挡内容

**技术亮点**:
- 纯前端实现，无需后端支持
- CSS 动画优先，性能优秀（<1MB 内存）
- 智能休眠，节省资源
- 离线时间计算，真实成长体验

**技术实现**:
- 文件: `static/JS/pet.js`, `static/CSS/pet.css`
- 存储: localStorage
- 动画: CSS Keyframes + JavaScript

---

### 5. 品种管理
**功能描述**: 完整的狗狗品种 CRUD 操作

**功能列表**:
- 📋 品种列表展示（分页、搜索、筛选）
- ➕ 添加新品种（管理员权限）
- ✏️ 编辑品种信息
- 🗑️ 删除品种
- ✅ 数据验证（名称唯一性、必填字段）

**技术实现**:
- 路由: `/admin/breeds`
- API: `/api/breeds` (GET, POST, PUT, DELETE)
- 权限控制: Flask-Login + 角色验证

---

### 6. 用户认证系统
**功能描述**: 安全的用户注册、登录、权限管理

**核心功能**:
- 👤 用户注册（密码强度验证）
- 🔑 用户登录（Session + JWT Token）
- 🚪 用户登出
- 🔒 权限控制（普通用户 vs 管理员）
- ⭐ 收藏功能

**技术实现**:
- 认证框架: Flask-Login 0.6.3
- Token: PyJWT 2.10.1
- 密码加密: Werkzeug security
- Session 管理: Flask Session

---

### 7. RESTful API
**功能描述**: 标准化的 API 接口，支持移动端和第三方集成

**API 列表**:
- `GET /api/breeds` - 获取品种列表
- `GET /api/breeds/{id}` - 获取单个品种
- `POST /api/breeds` - 添加品种（需管理员权限）
- `PUT /api/breeds/{id}` - 更新品种
- `DELETE /api/breeds/{id}` - 删除品种
- `POST /api/upload-data` - 上传数据文件
- `POST /api/generate-chart` - 生成图表
- `POST /api/export-data` - 导出数据
- `POST /api/save_pet_logs` - 保存宠物日志

**技术实现**:
- Blueprint: `api_bp`
- 响应格式: JSON
- 认证: JWT Token / Session
- 文档: Swagger/OpenAPI（待完善）

## 🚀 快速开始

### 环境要求
- Python 3.10+
- MySQL 8.0+
- pip (Python 包管理器)

### 安装步骤

#### 1. 克隆项目
```bash
git clone https://github.com/Cguomei/FastApiProject.git
cd FastApiProject
```

#### 2. 创建虚拟环境（推荐）
```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

#### 3. 安装依赖
```bash
pip install -r requirements.txt
```

#### 4. 配置环境变量
```bash
# 复制示例文件
copy .env.example .env  # Windows
cp .env.example .env    # Linux/Mac

# 编辑 .env 文件，填写实际配置
```

**.env 配置示例**:
```env
FLASK_ENV=development
SECRET_KEY=your-secret-key
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=dog
JWT_SECRET_KEY=jwt-secret-key
```

#### 5. 初始化数据库
```bash
python init_db.py
```

#### 6. 启动应用
```bash
python app.py
```

#### 7. 访问应用
打开浏览器访问: `http://localhost:5000`

## 📊 数据库设计

### 核心表结构

#### users (用户表)
| 字段 | 类型 | 说明 |
|------|------|------|

| username | VARCHAR(50) | 用户名（唯一） |

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |

## 🔒 安全特性

### 1. 密码安全
- 使用 Werkzeug 的 `generate_password_hash` 加密
- 密码长度至少 6 位
- 支持任意字符类型

### 2. 认证安全
- Session 认证（Web 端）
- JWT Token 认证（API 端）
- CSRF 保护（WTForms）

### 3. 权限控制
- 基于角色的访问控制（RBAC）
- 管理员权限验证
- API 接口权限检查

### 4. 敏感信息保护
- `.env` 文件不提交到 Git
- 提供 `.env.example` 模板
- `.gitignore` 正确配置
