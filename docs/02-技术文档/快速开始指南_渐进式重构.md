## 🎨 Alpine.js 学习路径

### 第 1 步：理解核心概念
阅读官方文档：https://alpinejs.dev/start-here

**核心指令**:
- `x-data`: 定义响应式数据
- `x-text`: 文本绑定
- `x-html`: HTML 绑定
- `:class` / `:style`: 动态样式
- `@click` / `@keyup`: 事件监听
- `x-show` / `x-if`: 条件渲染
- `x-for`: 列表渲染
- `x-model`: 双向绑定
- `x-ref`: DOM 引用
- `x-init`: 生命周期

### 第 2 步：对比 jQuery vs Alpine.js

**jQuery 方式（旧）**:
```javascript
// 手动操作 DOM
$('#pet-name').text('团团');
$('#feed-btn').on('click', function() {
    state.hunger -= 20;
    $('#hunger-bar').css('width', state.hunger + '%');
});
```

**Alpine.js 方式（新）**:
```html
<!-- 声明式绑定 -->
<div x-data="{ petName: '团团', hunger: 30 }">
    <span x-text="petName"></span>
    <button @click="hunger -= 20">喂食</button>
    <div :style="`width: ${hunger}%`"></div>
</div>
```

## 🔧 常见问题

### Q1: 健康检查返回 503 错误？
**原因**: 数据库连接失败  
**解决**: 
```bash
# 检查数据库服务
mysql -u doguser -p

# 检查环境变量
cat .env | grep DB_

# 重启应用
python app.py
```

### Q2: Alpine.js 不生效？
**原因**: CDN 加载失败或脚本顺序错误  
**解决**:
1. 检查浏览器控制台是否有错误
2. 确认 Alpine.js 在 pet-alpine.js 之前加载
3. 清除浏览器缓存（Ctrl+Shift+R）

### Q3: 测试失败？
**原因**: 数据库未初始化或缺少测试数据  
**解决**:
```bash
# 重新初始化数据库
python init_db.py

# 清理测试数据
python -m pytest Test/ --clean
```

### Q4: 宠物不显示？
**原因**: localStorage 中保存了隐藏状态  
**解决**:
```javascript
// 浏览器控制台执行
localStorage.removeItem('virtualPetState');
location.reload();
```
