## 📋 功能概述

为了解决**数据过时**问题，开发了完整的数据新鲜度监控与自动更新系统：

### ✅ 核心功能

1. **数据新鲜度检查**
   - 实时检测数据是否过期
   - 分级预警（新鲜/警告/过期/严重过期）
   - 可视化状态显示

2. **品种级监控**
   - 每个品种独立的新鲜度追踪
   - TOP20品种实时监控
   - 按过期时间排序

3. **智能建议**
   - 根据数据状态提供更新建议
   - 自动判断是否需要更新
   - 优先级评估

4. **旧数据清理**
   - 可配置保留天数
   - 自动清理过期数据
   - 释放数据库空间

## 🐛 常见问题

### Q1: 为什么显示"无数据"？

**A:** 数据库中还没有爬取过数据。请访问 `/crawler` 页面执行首次爬取。

### Q2: 如何调整新鲜度阈值？

**A:** 修改 `utils/data_freshness.py` 中的配置：
```python
self.freshness_threshold_hours = 24  # 改为其他值
self.warning_threshold_hours = 48
self.critical_threshold_hours = 72
```

### Q3: 自动更新何时触发？

**A:** 当前需要手动触发。可以配置定时任务：
```python
@scheduler.task('interval', hours=6)
def auto_update():
    if auto_scheduler.should_auto_update():
        # 触发爬虫
```
