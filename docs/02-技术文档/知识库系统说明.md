# 📚 AI助手知识库系统说明

## 概述

知识库系统是AI助手的**智能缓存层**，用于：
1. **快速响应**常见问题（无需调用模型）
2. **减少404错误**（避免模型未启动时的问题）
3. **提高稳定性**（标准答案保证质量）
4. **降低成本**（减少模型调用次数）

---

## 🎯 工作原理

```
用户提问
   ↓
问题分类器识别类型
   ↓
查询本地知识库 ← 优先使用
   ↓
命中？
 ├─ YES → 直接返回标准答案（毫秒级响应）✅
 └─ NO  → 调用Ollama模型生成回答（秒级响应）
           ↓
        学习新知识（可选加入知识库）
```

## 🔧 技术实现

### 文件结构

```
utils/

└── knowledge_base.json        # 知识库数据文件（自动生成）

├── ai_chat.html               # 聊天页面（显示知识库状态）
└── knowledge_base.html        # 知识库管理页面
```

### 核心API

#### 1. 查询知识库
```python
from utils.knowledge_base import get_knowledge_base

kb = get_knowledge_base()
result = kb.search("泰迪有什么特点？", "breed_info")

if result:
    print(result['answer'])  # 返回标准答案
else:
    print("未找到，需要调用模型")
```

#### 2. 添加新知识
```python
# 从模型回答中学习
kb.add_from_model_response(
    question="如何给狗狗洗澡？",
    answer="详细的洗澡步骤...",
    category="general_qa",
    confidence=0.85
)
```

#### 3. 获取统计信息
```python
stats = kb.get_stats()
print(f"总知识数: {stats['total_knowledge']}")
print(f"命中率: {stats['hit_rate']}")
```

## 🛠️ 维护和扩展

### 添加新知识

#### 方法1: 修改JSON文件

编辑 `data/knowledge_base.json`：

```json
{
  "knowledge": {
    "如何训练狗狗握手": {
      "question_patterns": ["怎么.*握手", "训练.*握手"],
      "answer": "训练步骤：\n1. 准备零食\n2. ...",
      "category": "general_qa",
      "confidence": 0.9,
      "created_at": "2026-05-09T...",
      "usage_count": 0
    }
  }
}
```

#### 方法2: 通过代码添加

```python
from utils.knowledge_base import get_knowledge_base

kb = get_knowledge_base()

kb.add_knowledge("训练握手", {
    "question_patterns": ["怎么.*握手", "训练.*握手"],
    "answer": "详细步骤...",
    "category": "general_qa",
    "confidence": 0.9,
    "created_at": datetime.now().isoformat(),
    "usage_count": 0
})

kb.save()
```

### 清理无用知识

```python
kb = get_knowledge_base()

# 清理90天未使用的知识
removed = kb.clear_unused(days=90)
print(f"清理了 {removed} 条知识")
```

## 📈 性能提升

### 响应时间对比

| 场景 | 之前 | 现在 | 提升 |
|------|------|------|------|
| 知识库命中 | - | <100ms | ⚡⚡⚡ |
| 模型调用 | 3-5秒 | 3-5秒 | - |

## 🎓 最佳实践

### 1. 优先使用知识库
- 常见问题都应该有知识库条目
- 确保答案准确、完整、友好

### 2. 定期更新知识
- 根据用户反馈优化答案
- 添加新的热门问题
- 清理过时内容

### 3. 监控命中率
- 目标：>80% 命中率
- 低于70% 需要补充知识
- 高于90% 说明覆盖良好

### 4. 保持知识质量
- 答案要简洁明了
- 使用格式化（粗体、列表）
- 包含实用建议

## 🔍 故障排查

### Q1: 知识库没有生效？

**检查**：
1. 日志中是否有"知识库命中"？
2. `data/knowledge_base.json` 是否存在？
3. 问题模式是否匹配？

**解决**：
```bash
# 查看日志
type log\ai_assistant.log | findstr "知识库"
```

### Q2: 命中率太低？

**原因**：
- 知识覆盖不足
- 问题模式不够全面

**解决**：
1. 分析未命中的问题
2. 添加对应的知识条目
3. 扩充question_patterns

### Q3: 如何查看哪些知识最常用？

```python
kb = get_knowledge_base()

# 按使用次数排序
sorted_kb = sorted(
    kb.knowledge.items(),
    key=lambda x: x[1].get('usage_count', 0),
    reverse=True
)

for key, item in sorted_kb[:10]:
    print(f"{key}: {item['usage_count']}次")
```
