1. 数据质量报告展示
--------------------------------------------------------------------------------
功能描述:
  在用户上传数据后，自动显示后端返回的数据质量校验报告。

UI设计:
  - 位置: 上传成功信息下方
  - 样式: Bootstrap Alert组件
  - 图标: 
    * ✅ 绿色 - 无问题
    * ⚠️ 黄色警告 - warning级别
    * ❌ 红色错误 - error级别
    * ℹ️ 蓝色信息 - info级别

HTML结构:
```html
<!-- 数据质量报告 -->
<div id="qualityReport" class="collapse mt-3"></div>
```

JavaScript实现:
```javascript
function displayQualityReport(report) {
    const reportDiv = document.getElementById('qualityReport');
    
    if (!report.issues || report.issues.length === 0) {
        // 无问题 - 显示成功消息
        reportDiv.innerHTML = '<div class="alert alert-success">
            <i class="bi bi-check-circle-fill me-2"></i>
            <strong>✅ 数据质量良好</strong>，无发现问题
        </div>';
        reportDiv.classList.remove('collapse');
        return;
    }
    
    // 有问题 - 列出所有问题
    let html = '<div class="alert alert-warning">
        <strong><i class="bi bi-exclamation-triangle-fill me-2"></i>
        ⚠️ 数据质量报告</strong>
        <ul class="mb-0 mt-2">';
    
    report.issues.forEach(issue => {
        const icon = issue.severity === 'error' ? '❌' : 
                    issue.severity === 'warning' ? '⚠️' : 'ℹ️';
        const color = issue.severity === 'error' ? 'text-danger' : 
                     issue.severity === 'warning' ? 'text-warning' : 'text-info';
        html += `<li class="${color}">${icon} ${issue.message}</li>`;
    });
    
    html += '</ul></div>';
    reportDiv.innerHTML = html;
    reportDiv.classList.remove('collapse');
}
```

调用时机:
  在uploadFile()函数中，收到后端响应后立即调用：
```javascript
if (result.quality_report) {
    displayQualityReport(result.quality_report);
}
```

视觉效果:

  - 严重错误: 红色文字标识

JavaScript实现:
```javascript
async function exportData(format) {
    if (!uploadedData || uploadedData.length === 0) {
        alert('没有数据可导出');
        return;
    }
    
    try {
        const response = await fetch('/api/export-data', {
            method: 'POST',
            headers: {'Content-Type': 'application/json'},
            body: JSON.stringify({
                format: format,
                filename: '自定义分析数据',
                data: uploadedData
            })
        });
        
        if (!response.ok) {
            throw new Error('导出失败');
        }
        
        // 触发浏览器下载
        const blob = await response.blob();
        const url = window.URL.createObjectURL(blob);
        const a = document.createElement('a');
        a.href = url;
        a.download = `自定义分析数据.${format === 'excel' ? 'xlsx' : 'csv'}`;
        document.body.appendChild(a);
        a.click();
        window.URL.revokeObjectURL(url);
        document.body.removeChild(a);
        
    } catch (error) {
        alert('导出失败: ' + error.message);
    }
}
```

技术要点:
  1. 使用async/await处理异步请求
  2. Fetch API发送POST请求到后端
  3. Blob对象处理二进制数据
  4. 动态创建<a>标签触发下载
  5. 下载完成后清理URL对象和DOM元素

用户体验:
  - 点击按钮立即开始下载
  - 文件名自动包含时间戳
  - 支持Excel和CSV两种格式
  - 错误时显示友好提示

JavaScript实现:
```javascript
async function exportChartData() {
    try {
        // 根据当前页面路径确定数据来源API
        const path = window.location.pathname;
        let apiUrl = '';
        
        if (path.includes('/chart/scatter')) {
            apiUrl = '/api/breeds';
        } else if (path.includes('/chart/line')) {
            apiUrl = '/api/breeds';
        } else if (path.includes('/chart/bar')) {
            apiUrl = '/api/breeds';
        } else if (path.includes('/chart/hist')) {
            apiUrl = '/api/food';
        } else if (path.includes('/chart/funnel')) {
            apiUrl = '/api/breeds';
        } else if (path.includes('/chart/map')) {
            apiUrl = '/api/breeds';
        }
        
        if (!apiUrl) {
            alert('无法确定数据来源');
            return;
        }
        
        // 获取数据
        const response = await fetch(apiUrl);
        const data = await response.json();
        
        if (!data || data.length === 0) {
            alert('没有数据可导出');
            return;
        }
        
        // 调用导出API
        const exportResponse = await fetch('/api/export-data', {
            method: 'POST',
            headers: {'Content-Type': 'application/json'},
            body: JSON.stringify({
                format: 'excel',
                filename: '{{ chart_title }}',
                data: data
            })
        });
        
        if (!exportResponse.ok) {
            throw new Error('导出失败');
        }
        
        // 触发下载
        const blob = await exportResponse.blob();
        const url = window.URL.createObjectURL(blob);
        const a = document.createElement('a');
        a.href = url;
        a.download = '{{ chart_title }}.xlsx';
        document.body.appendChild(a);
        a.click();
        window.URL.revokeObjectURL(url);
        document.body.removeChild(a);
        
    } catch (error) {
        alert('导出失败: ' + error.message);
    }
}
```

智能路由:
  根据不同的图表页面，自动选择对应的数据API：
  - 散点图/折线图/柱状图 → /api/breeds (品种数据)
  - 直方图 → /api/food (狗粮数据)
  - 漏斗图/地图 → /api/breeds (品种数据)

优势:
  - 用户无需手动选择数据源
  - 自动匹配正确的数据集
  - 统一的导出体验

九、与后端接口对接
================================================================================

1. POST /api/upload-data
   请求: FormData (file)
   响应: {success, columns, row_count, data, quality_report}
   用途: 上传数据并获取质量报告

2. POST /api/generate-chart
   请求: {chart_type, x_column, y_column, title, data}
   响应: {success, chart_html, message, data_points}
   用途: 生成自定义图表

3. POST /api/export-data
   请求: {format, filename, data}
   响应: 文件下载 (Blob)
   用途: 导出数据为Excel/CSV

4. GET /api/breeds
   请求: 无
   响应: [{id, breed_name, avg_life_years, size_category, popularity}]
   用途: 获取品种数据用于导出

5. GET /api/food
   请求: 无
   响应: 狗粮列表数据
   用途: 获取狗粮数据用于导出
