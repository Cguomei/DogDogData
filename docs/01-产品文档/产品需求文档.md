## 一、产品概述

### 1.1 产品背景

随着宠物经济的快速发展，宠物狗市场数据呈现出爆炸式增长。为了帮助宠物店、繁育者和消费者更好地了解市场动态，需要一款专业的数据分析工具来展示狗狗品种信息、价格分布、市场趋势等关键数据。

### 1.2 产品定位

本产品是一个基于 Web 的狗狗数据可视化分析平台，提供：

- 多维度数据图表展示
- 品种信息管理
- 狗粮数据查询
- 用户权限管理

### 1.3 目标用户

| 用户类型   | 使用场景        | 核心需求        |
| ------ | ----------- | ----------- |
| 宠物店经营者 | 了解市场行情、定价策略 | 价格分布、热门品种统计 |
| 宠物繁育者  | 选育决策、市场分析   | 品种信息、血统级别分布 |
| 潜在购买者  | 选购参考、价格对比   | 价格区间、产地信息   |
| 数据分析师  | 数据挖掘、趋势分析   | 完整数据导出、多维分析 |

### 1.4 产品目标

#### 业务目标

- 提供准确的狗狗市场数据统计
- 支持科学的购买和繁育决策
- 建立完善的品种信息管理体系

#### 技术目标

- 响应时间：首页 < 2 秒，API < 1 秒
- 数据准确性：99.9%
- 系统可用性：99%
- 安全性：通过基本安全测试

## 二、功能需求

### 2.1 功能架构

```
狗狗数据分析系统
├── 用户认证模块
│   ├── 用户注册
│   ├── 用户登录
│   ├── 用户登出
│   └── 权限管理
├── 数据看板模块
│   ├── 首页统计
│   ├── 图表展示
│   └── 数据汇总
├── 品种管理模块
│   ├── CRUD 操作
│   ├── 批量导入
│   └── 数据验证
├── 狗粮数据模块
│   ├── 品牌统计
│   ├── 价格分析
│   └── 产地分布
└── 系统管理模块
    ├── 定时任务
    ├── 数据缓存
    └── 错误处理
```

### 2.2 详细功能说明

#### F1: 用户认证模块

**F1.1 用户注册**

- **功能描述**: 新用户创建账户
- **输入**: 用户名（3-20 字符）、密码（≥6 位，不限字符类型）
- **处理**: 
  - 验证用户名格式（中文、字母、数字、下划线）
  - 检查用户名唯一性
  - 密码加密存储
  - 密码强度验证（至少 6 位）
- **输出**: 注册成功跳转登录页
- **优先级**: P0

**F1.2 用户登录**

- **功能描述**: 已注册用户登录系统
- **输入**: 用户名、密码
- **处理**: 
  - 验证用户凭证
  - 创建会话
  - 不明确指出错误原因（防枚举）
- **输出**: 登录成功跳转首页
- **优先级**: P0

**F1.3 权限管理**

- **功能描述**: 基于角色的访问控制
- **角色定义**:
  - 管理员：所有操作权限
  - 普通用户：查看权限
- **优先级**: P1

---

#### F2: 数据看板模块

**F2.1 首页统计**

- **功能描述**: 展示核心业务指标
- **统计项**:
  - 狗狗总数
  - 平均价格
  - 店铺总数
  - 品种总数
  - TOP5 热门品种
  - TOP5 热门店铺
  - 价格区间分布
  - 血统级别比例
- **优先级**: P0

**F2.2 价格散点图**

- **功能描述**: 展示价格与数量关系
- **X 轴**: 价格（元）
- **Y 轴**: 数量（只）
- **交互**: 悬停显示详细信息
- **优先级**: P1

**F2.3 体重折线图**

- **功能描述**: 展示不同体重区间的分布趋势
- **分组**: 0-1kg, 1-2kg, 2-5kg, 5-10kg, 10-20kg, 20kg 以上
- **标记**: 最大值、最小值
- **优先级**: P1

**F2.4 级别柱状图**

- **功能描述**: 对比各品种的宠物级和血统级数量
- **维度**: 按品种分组
- **系列**: 宠物级、血统级
- **优先级**: P2

**F2.5 TOP10 直方图**

- **功能描述**: 展示最热门的 10 种狗狗和店铺
- **内容**: 
  - 狗狗品种 TOP10
  - 店铺销量 TOP10
- **特性**: 支持缩放
- **优先级**: P2

**F2.6 价格段漏斗图**

- **功能描述**: 展示各价格区间的分布情况
- **区间**: 
  - 0-2.5k
  - 2.5k-5k
  - 5k-7.5k
  - 7.5k-1w
  - 1w-2w
  - 2w 以上
- **优先级**: P2

**F2.7 世界地图**

- **功能描述**: 展示狗狗产地分布
- **数据**: 各国/地区狗狗数量
- **视觉**: 颜色深浅表示数量
- **翻译**: 中文地名自动翻译为英文
- **缓存**: 1 小时
- **优先级**: P2

---

#### F3: 品种管理模块

**F3.1 查看品种列表**

- **功能描述**: 展示所有犬种信息
- **字段**: ID、品种名、平均寿命、体型、人气值
- **格式**: JSON
- **权限**: 公开
- **优先级**: P1

**F3.2 添加品种**

- **功能描述**: 创建新的犬种记录
- **必填字段**: 品种名（2-100 字符）
- **可选字段**: 
  - 平均寿命（0-100 岁）
  - 体型类别（小型/中型/大型/超大型）
  - 人气值（0-1000）
- **验证**: 
  - 名称唯一性检查
  - XSS 过滤
  - 数据类型和范围验证
- **权限**: 管理员
- **优先级**: P1

**F3.3 编辑品种**

- **功能描述**: 更新现有犬种信息
- **验证**: 同添加功能（排除自身名称检查）
- **权限**: 管理员
- **优先级**: P1

**F3.4 删除品种**

- **功能描述**: 删除犬种记录
- **权限**: 管理员
- **优先级**: P1

**F3.5 批量导入**

- **功能描述**: 从 CSV/Excel 文件批量导入品种数据
- **支持格式**: .csv, .xlsx, .xls
- **列映射**: 支持中英文列名
- **错误处理**: 单条失败不影响其他数据
- **返回**: 成功数、失败数、失败详情
- **权限**: 管理员
- **优先级**: P2

---

#### F4: 狗粮数据模块

**F4.1 狗粮看板**

- **功能描述**: 展示狗粮统计数据
- **统计项**:
  - 品牌总数
  - 平均价格
  - 产地分布 TOP5
  - 价格区间分布
- **优先级**: P2

**F4.2 狗粮明细**

- **功能描述**: 查看狗粮详细数据
- **字段**: 品牌名、价格、产地
- **限制**: 最多返回 100 条
- **优先级**: P3

---

#### F5: 系统功能

**F5.1 数据缓存**

- **功能描述**: 提高图表加载速度
- **策略**: 地图数据缓存 1 小时
- **实现**: Flask-Caching
- **优先级**: P2

**F5.2 定时任务**

- **功能描述**: 自动更新数据汇总表
- **频率**: 每 6 小时
- **内容**: 计算各项统计数据
- **实现**: Flask-APScheduler
- **优先级**: P2

**F5.3 安全防护**

- **功能描述**: 保护系统安全
- **措施**:
  - CSRF 保护（表单提交）
  - SQL 注入防护（ORM 参数化查询）
  - XSS 防护（输入过滤、输出转义）
  - 密码加密存储
  - 安全响应头
- **优先级**: P0

## 三、非功能需求

### 3.1 性能需求

| 指标       | 要求      | 测量方法      |
| -------- | ------- | --------- |
| 首页响应时间   | < 2 秒   | 10 次请求平均值 |
| API 响应时间 | < 1 秒   | 10 次请求平均值 |
| 图表页面加载   | < 5 秒   | 从请求到渲染完成  |
| 并发用户数    | ≥ 10    | 同时在线不卡顿   |
| 数据库查询    | < 500ms | 复杂统计查询    |

### 3.2 安全需求

| 安全项     | 要求   | 验证方法        |
| ------- | ---- | ----------- |
| SQL 注入  | 完全免疫 | 渗透测试        |
| XSS 攻击  | 完全阻止 | 脚本注入测试      |
| CSRF 攻击 | 有效防护 | 跨站请求测试      |
| 密码泄露    | 零容忍  | 数据库检查       |
| 会话劫持    | 防止   | Cookie 属性检查 |

### 3.3 可用性需求

- **系统可用性**: 99%（每月宕机时间 < 7.2 小时）
- **数据准确性**: 99.9%
- **错误恢复**: 关键错误可回滚
- **日志记录**: 完整记录异常

### 3.4 兼容性需求

| Python  | 3.8+ |
| MySQL   | 8.0+ |
| Chrome  | 最新版  |
