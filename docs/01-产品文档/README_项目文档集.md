# 狗狗数据分析系统 - 项目文档集

## 📋 项目概述

**项目名称**: 狗狗数据分析系统  
**当前版本**: v2.4.0  
**创建日期**: 2026-03-23  
**更新日期**: 2026-03-24  
**技术栈**: Flask + PyECharts + MySQL  

本项目是一个基于 Web 的狗狗数据可视化分析平台，提供数据展示、品种管理、图表分析等功能，帮助宠物店、繁育者和消费者了解市场动态。

## 🚀 快速开始

### 环境要求
- Python 3.9+
- MySQL 8.0+
- Chrome/Firefox/Edge浏览器

### 安装步骤
```bash
# 1. 克隆代码
git clone <repository-url>
cd fastApiProject

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，配置数据库连接

# 4. 初始化数据库
python -c "from app import app; from models import db; \
with app.app_context(): db.create_all()"

# 5. 创建管理员
python -c "from app import app, db; from models import User; \
with app.app_context(): \
    admin = User(username='admin', role='admin'); \
    admin.set_password('123456'); \
    db.session.add(admin); \
    db.session.commit()"

# 6. 启动服务
python app_new.py
```

访问：http://localhost:5000

## 🛡️ 安全特性

1. **CSRF 保护**
   - Flask-WTF 集成
   - Token 有效期 1 小时
   - 表单提交强制验证

2. **XSS 防护**
   - 输入过滤（HTML 标签）
   - 输出转义（Jinja2）
   - 响应头设置

3. **SQL 注入防护**
   - ORM 参数化查询
   - 禁止字符串拼接

4. **会话安全**
   - HttpOnly Cookie
   - 密码加密存储
   - Session 管理
