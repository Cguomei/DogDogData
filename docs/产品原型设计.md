## 一、设计概述

### 1.1 设计理念

- **简洁直观**: 用户可以快速找到所需功能
- **数据驱动**: 突出数据可视化，图表清晰易懂
- **响应式布局**: 适配不同屏幕尺寸
- **一致性**: 统一的交互和视觉风格

### 1.2 设计风格

- **主色调**: 蓝色系（专业、可靠）
- **辅助色**: 绿色（成功）、红色（警告）、橙色（提醒）
- **字体**: 系统默认无衬线字体
- **图标**: 简洁线条风格

## 二、页面原型

### 2.1 登录页面 (login.html)

#### 页面布局

```
┌─────────────────────────────────────┐
│                                     │
│         狗狗数据分析系统              │
│                                     │
│    ┌─────────────────────────┐     │
│    │  用户名：[          ]   │     │
│    │  密码：  [          ]   │     │
│    │  [ ] 记住我             │     │
│    │  [   登录   ]           │     │
│    │  还没有账号？注册      │     │
│    └─────────────────────────┘     │
│                                     │
└─────────────────────────────────────┘
```

#### 交互说明

1. **表单验证**
   - 用户名不能为空
   - 密码不能为空
   - 实时显示错误提示

2. **提交行为**
   - 点击登录后显示加载动画
   - 成功后跳转到首页或之前访问的页面
   - 失败后显示错误消息（不区分用户名或密码错误）

3. **CSRF 保护**
   - 表单包含隐藏的 CSRF token 字段

## 三、交互规范

### 3.1 按钮状态

| 状态 | 样式 | 说明 |
|-----|------|------|
| 默认 | 蓝色背景，白色文字 | 正常可点击状态 |
| 悬停 | 深蓝色背景 | 鼠标移入时 |

| 加载 | 蓝色 + 旋转图标 | 处理中状态 |

## 六、图标使用

### 6.1 功能图标

- 编辑：✏️ 或 FontAwesome `fa-edit`
- 删除：🗑️ 或 FontAwesome `fa-trash`
- 添加：➕ 或 FontAwesome `fa-plus`
- 搜索：🔍 或 FontAwesome `fa-search`
- 导入：📥 或 FontAwesome `fa-download`
- 导出：📤 或 FontAwesome `fa-upload`

### 6.2 状态图标

- 成功：✓ 或 FontAwesome `fa-check`
- 错误：✗ 或 FontAwesome `fa-times`
- 警告：⚠️ 或 FontAwesome `fa-exclamation-triangle`
- 信息：ℹ️ 或 FontAwesome `fa-info-circle`

## 七、交互动画

### 7.1 过渡动画

- **淡入淡出**: opacity 0 → 1 (0.3s)
- **滑动**: transform translateY (0.3s)
- **缩放**: scale 0.95 → 1 (0.2s)

### 7.2 微交互

- 按钮悬停：轻微放大 (scale 1.02)
- 卡片悬停：阴影加深
- 数据更新：数字滚动效果

## 八、可用性原则

### 8.1 F 型浏览模式

重要信息放在页面左侧和顶部

### 8.2 三次点击原则

任何功能在三次点击内可达

### 8.3 一致性

- 相同操作有相同反馈
- 相似页面有相似布局
- 统一术语使用

### 8.4 容错性

- 危险操作需要确认
- 提供撤销功能（如可能）
- 清晰的错误提示和恢复指引
