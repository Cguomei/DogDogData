## 🎯 项目概述

### 项目定位
一个基于 Flask 的宠物狗狗数据可视化平台，提供数据分析、AI智能助手、图表展示等功能。

### 核心价值
1. **数据可视化**: 8项核心指标 + 6种图表类型
2. **AI智能助手**: 基于知识库的智能问答系统（支持游客模式）
3. **用户交互**: 网页小宠物陪伴功能
4. **完整测试**: 71+自动化测试用例，100%通过率

### 技术特点
- ✅ 前后端分离架构（Flask后端 + HTML/JS前端）
- ✅ RESTful API设计
- ✅ 完善的测试体系（pytest + Playwright）
- ✅ CI/CD配置（Jenkins）
- ✅ 多语言支持（国际化）

## 🤖 AI助手系统详解

### 核心特性

#### 1. 知识库优先策略
```python
# 查询流程
用户问题 → 问题分类 → 知识库检索 → 
  ├─ 命中 → 返回知识库答案（标注来源）
  └─ 未命中 → 调用Ollama模型 → 返回模型答案
```

**知识库位置**: `data/knowledge_base.json`  
**知识库类别**:
- breed_info（品种信息）
- price_query（价格查询）
- care_tips（养护建议）
- recommendation（推荐）
- comparison（对比）

#### 2. 问题分类系统
```python
classify_question(user_message) → {
    'type': 'breed_info' | 'price_query' | 'recommendation' | ...,
    'params': {'breed': '金毛', ...}
}
```

#### 3. 上下文记忆
- 保存最近10轮对话
- 自动关联历史消息
- 支持多轮追问

#### 4. 游客模式 ⭐亮点
- ✅ 无需登录即可使用
- ✅ 自动创建guest用户账户
- ✅ 会话通过localStorage隔离
- ⚠️ 数据库层面共享guest账户
- ⚠️ 不同设备会话不互通

**实现细节**:
```python
# 动态获取或创建guest用户
guest_user = User.query.filter_by(username='guest').first()
if not guest_user:
    guest_user = User(username='guest', password_hash=..., role='guest')
    db.session.add(guest_user)
    db.session.commit()

user_id_for_session = guest_user.id if is_guest else current_user.id
```

#### 5. 自动学习机制
- 模型生成的回答自动加入知识库
- 提高后续命中率
- 减少模型调用成本

### Ollama模型配置
```python
# config.py
MODEL_TYPE = 'ollama'
LOCAL_MODEL_CONFIG = {
    'ollama': {
        'base_url': 'http://localhost:11434',
        'model': 'qwen2.5:7b',  # 根据实际安装的模型调整
        'timeout': 30
    }
}
```

**启动Ollama**:
```bash
ollama serve
ollama pull qwen2.5:7b
```

## 📌 重要提醒

### ⚠️ 注意事项

1. **环境变量**: `.env`文件包含敏感信息，不要提交到Git
2. **数据库备份**: 定期备份MySQL数据库
3. **Ollama服务**: AI助手依赖Ollama，确保服务正常运行
4. **测试先行**: 修改代码前先运行测试，确保不破坏现有功能
5. **Git提交**: 每次修改后立即提交，保持版本清晰

### 🔐 安全建议

1. **生产环境**: 必须使用HTTPS
2. **密码策略**: 强制复杂密码
3. **SQL注入**: 使用ORM参数化查询
4. **XSS防护**: 前端转义用户输入
5. **CSRF保护**: Flask-WTF已启用

### 🚀 性能建议

1. **缓存策略**: 合理使用Flask-Caching
2. **数据库索引**: 为常用查询字段添加索引
3. **图片优化**: 压缩图片，使用CDN
4. **异步处理**: 耗时操作使用APScheduler
5. **监控告警**: 使用Prometheus监控系统
