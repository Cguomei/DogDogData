## ⚠️ 升级注意事项

### 1. 环境变量配置

在`.env`文件中添加或确认以下配置：

```bash
# AI助手配置
MODEL_TYPE=ollama
LOCAL_MODEL_URL=http://localhost:11434
LOCAL_MODEL_NAME=qwen2.5:1.5b

# Redis配置（注意@符号的URL编码）
REDIS_URL=redis://:123456%40ch@localhost:6379
```

### 2. 依赖安装

```bash
pip install -r requirements.txt
```

### 3. 启动Ollama

```bash
# 启动Ollama服务
ollama serve

# 拉取模型（如果还没有）
ollama pull qwen2.5:1.5b
```

### 4. 数据库迁移

```bash
# 运行迁移脚本（如果需要）
python init_db.py
```

### 5. 重启服务

根据实际部署方式重启Flask应用。
