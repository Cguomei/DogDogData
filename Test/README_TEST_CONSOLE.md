# 🧪 智能测试控制台 - 快速开始

## ⚡ 30秒快速上手

### 步骤1: 启动控制台

**方式A - 双击启动（推荐）**
```
双击文件: Test\start_test_console.bat
```

**方式B - 命令行启动**
```bash
python Test/test_server.py
```

### 步骤2: 打开浏览器

浏览器会自动打开，或手动访问：
```
http://localhost:5555
```

### 步骤3: 开始测试

点击任意按钮即可开始测试：
- 🚀 **全面测试** - 运行所有测试
- 🔌 **API测试** - 只测试API
- 📊 **覆盖率报告** - 查看代码覆盖情况

## 🔧 配置说明

### 修改端口

编辑 `Test/test_server.py` 最后一行：
```python
app.run(host='0.0.0.0', port=5555, debug=True)
#                                    ^^^^ 修改这里
```

### 添加新测试

1. 在 `test_server.py` 添加路由
2. 在 `test_console.html` 添加按钮
3. 编写对应的JavaScript函数

详见 [TEST_CONSOLE_GUIDE.md](TEST_CONSOLE_GUIDE.md)
