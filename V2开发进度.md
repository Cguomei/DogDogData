### 4. 上下文对话支持

需要在调用模型时传入历史消息：

```python
def call_local_model_with_context(messages: list, session_id: int):
    """带上下文的模型调用"""
    # 获取最近10条历史消息
    history = ChatMessage.query.filter_by(
        session_id=session_id
    ).order_by(
        ChatMessage.created_at.desc()
    ).limit(10).all()
    
    # 反转顺序（从旧到新）
    history.reverse()
    
    # 构建完整的消息列表
    full_messages = [
        {"role": msg.role, "content": msg.content}
        for msg in history
    ]
    
    # 添加当前消息
    full_messages.extend(messages)
    
    # 调用模型
    return call_local_model(full_messages)
```
